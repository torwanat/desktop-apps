# AirQuality

AirQuality is a Tkinter-based utility that fetches and displays air quality measurements (PM2.5, PM10 and chemical values) from the Polish GIOS API.

Features
- Select city and measurement station
- View recent PM and chemical readings

Quick start
1. Install Python 3.7+ and `requests` (`pip install requests`).
2. Run `python airQuality.py` from the `AirQuality` folder.
