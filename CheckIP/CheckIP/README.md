# CheckIP

Console utility that inspects local network configuration and active connections. Shows hostname, IP addresses, network interfaces and basic DNS info.

Features
- Display local hostname and IPs
- List active TCP/UDP connections
- Show network interface details

Quick start
1. Open the `CheckIP.sln` in Visual Studio or run the project in the `CheckIP` folder.
2. Build and run the console application.

See `Program.cs` for entry point and implementation details.
