# Client (TCP Chat)

WPF TCP client application for simple chat and displaying received HTML content in a browser control.

Features
- Connect to TCP servers
- Send/receive messages asynchronously
- Render received content in embedded browser control

Quick start
1. Open `Client.sln` in Visual Studio.
2. Build and run the WPF project from the `Client` folder.

See `MainWindow.xaml.cs` for UI logic and networking code.
