# FileManager

A Qt-based file management utility (PySide6) with tree navigation, basic file operations and a simple text editor built into the UI.

Features
- Tree-based filesystem navigation
- Create, copy, move, rename and delete files/folders
- Inline text editor for quick edits

Quick start
1. Install Python 3.7+ and `PySide6` (`pip install PySide6`).
2. Run `main.py` from the `FileManager` folder.

See `main.py` and other modules for UI structure and operations.
