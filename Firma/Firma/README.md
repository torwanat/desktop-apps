# Firma

Console application demonstrating object-oriented management of company personnel and teams (employees, team members, team leaders, teams).

Features
- Create and manage `Osoba`, `CzlonekZespolu`, `KierownikZespolu` objects
- Organize employees into teams and clone records

Quick start
1. Open the `Firma.sln` in Visual Studio or run the `Firma` project.
2. Build and run the console application.

See project files (`Osoba.cs`, `Zespol.cs`) for implementation details.
