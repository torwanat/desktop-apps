# GuestBook

An ASP.NET Web Forms guest book that stores messages in `book.xml` and uses XSLT for presentation.

Features
- Submit and view guest messages
- XML-backed storage with XSLT transformation

Quick start
1. Open `GuestBook.sln` in Visual Studio.
2. Run the web project (IIS Express) and open the guest form.

See `GuestForm.aspx`, `View.aspx`, `book.xml` and `book.xslt` for implementation.
