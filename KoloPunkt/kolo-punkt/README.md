# KoloPunkt

Educational console application demonstrating points and circles, with routines to calculate area, perimeter and basic geometry operations.

Features
- `point` and `Circle` classes
- Compute circle area/perimeter and point operations

Quick start
1. Open `kolo-punkt.sln` in Visual Studio or run the `kolo-punkt` project.
2. Build and run the console application.

See `point.cs` and `Circle.cs` for source.
