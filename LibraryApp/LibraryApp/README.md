# LibraryApp

Web-based library management system built on ASP.NET Web Forms. Provides book cataloging and basic database-backed operations.

Features
- Manage book inventory (add/remove)
- Database-backed persistence (see SQL schema in repository root)

Quick start
1. Open `LibraryApp.sln` in Visual Studio.
2. Configure database connection in `Web.config` and publish or run locally.

See the project files in the `LibraryApp` folder and `library.sql` for schema details.
