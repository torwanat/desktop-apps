# MysqlManager

Tkinter-based helper for basic MySQL administration tasks: list/create/drop databases and connect using provided credentials.

Features
- Login and list databases
- Create and remove databases
- Simple GUI for common DB tasks

Quick start
1. Install Python 3.7+ and `mysql-connector-python` (`pip install mysql-connector-python`).
2. Run `main.py` from the `MysqlManager` folder.

See `database.py` and `layout.py` for implementation details.
