# Port Scanner

WPF port scanner utility that performs multi-threaded scanning of specified port ranges on target hosts and reports open ports.

Features
- Specify host and port ranges
- Multi-threaded scanning with real-time results

Quick start
1. Open `Port Scanner.sln` in Visual Studio.
2. Build and run the WPF project in the `Port Scanner` folder.

See `MainWindow.xaml.cs` for scanning logic.
