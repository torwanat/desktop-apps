# TODO

Lightweight task manager built with PySide6 and SQLite. Create, edit, and track tasks with deadlines and completion state.

Features
- Add, edit, delete tasks
- Deadlines with date/time picker
- Persistent storage using SQLite

Quick start
1. Install Python 3.7+ and `PySide6`.
2. Run `main.py` in the `TODO` folder.

See project files for DB schema and UI actions.
