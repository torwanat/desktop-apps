# TextEditor

Full-featured PySide6-based rich text editor with file operations and basic formatting (bold, italic, underline, fonts).

Features
- Open/create/save text files
- Rich text formatting and toolbar shortcuts

Quick start
1. Install Python 3.7+ and `PySide6` (`pip install PySide6`).
2. Run `main.py` from the `TextEditor` folder.

See the project files for toolbar actions and keyboard shortcuts.
